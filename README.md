&#x20;AI Agent for Automated Jira Issue Creation



📌 Project Overview



This project demonstrates an AI-powered automation workflow built using n8n. The workflow accepts user input, processes it using Generative AI, and automatically creates Jira issues through the Jira REST API.



The project showcases AI workflow automation, API integration, and low-code development using n8n.

🔄 Workflow



1\. User submits a request.

2\. AI analyzes the prompt.

3\. Required Jira fields are extracted.

4\. Jira REST API is invoked.

5\. Jira issue is created automatically.

6\. Success or failure response is returned.
